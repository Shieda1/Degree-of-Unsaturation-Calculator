How to calculate degree of unsaturation?
To calculate degree of unsaturation you should use the following formula:
degree of unsaturation = (2 (c) - H - x + N + 2) / 2
where:
c = number of carbon atoms
H = number of hydrogen atoms
x = number of halogen atoms
N = number of nitrogen atoms
